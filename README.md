# Etch-a-Sketch

A browser-based sketchpad built as part of **The Odin Project's** Foundations Course. This project focuses on DOM manipulation using JavaScript and creating dynamic layouts with CSS Grid and Flexbox.

## ✨ Features
- **Dynamic Grid:** Users can choose the size of the drawing board (up to 100x100).
- **Smooth Drawing:** Hover/drag over the cells to change their color.
- **Color Modes:** 
  - Monochrome (Black/Gray).
  - Rainbow (Randomly generated colors).
  - Progressive Darkening (Incremental opacity).
- **Clear Board:** Reset the canvas with a single click.

## 🛠️ Built With
- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
